f:case
======

This ViewHelper is part of the use case for f:switch. Where the value of this tag matches the value in the f:switch tag, 
the content of this tag will be rendered and output.

Properties
----------

#######################################

 :header-rows: 1

 - :Property,20:    Property
   :Datatype,20:    Variable type
   :Description,40: Description
   :Standard,10:    Standard
   :Mandatory,10:   Mandatory

 - :Property:    value
   :Datatype:    Mixed
   :Description: The value against which the value from the f:switch ViewHelper should be compared.
   :Standard:
   :Mandatory:   Yes
