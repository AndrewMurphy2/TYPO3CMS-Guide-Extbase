
Global properties of the HTML element
#####################################

.. t3-field-list-table::
 :header-rows: 1

 - :Property,20:    Property
   :Datatype,20:    Variable type
   :Description,40: Description
   :Standard,10:    Standard
   :Mandatory,10:   Mandatory

 - :Property:    class
   :Datatype:    String
   :Description: CSS class name for this HTML element
   :Standard:    NULL
   :Mandatory:   No

 - :Property:    dir
   :Datatype:    String
   :Description: The text direction. Allowed values are 'ltr' (left to right) and 'rtl' (right to left).
   :Standard:    NULL
   :Mandatory:   No

 - :Property:    id
   :Datatype:    String
   :Description: An identifier for the element, which must be unique within the current document.
   :Standard:    NULL
   :Mandatory:   No

 - :Property:    lang
   :Datatype:    String
   :Description: Language identifier for this HTML element, in line with RFC 1766.
   :Standard:    NULL
   :Mandatory:   No

 - :Property:    style
   :Datatype:    String
   :Description: Individual inline CSS styles, which should only be relevant to the current HTML element.
   :Standard:    NULL
   :Mandatory:   No

 - :Property:    title
   :Datatype:    String
   :Description: An advisory text, usually displayed when moving over the element with a mouse pointer.
   :Standard:    NULL
   :Mandatory:   No

 - :Property:    accesskey
   :Datatype:    String
   :Description: A definition of the key combination which will bring focus to this element.
   :Standard:    NULL
   :Mandatory:   No

 - :Property:    tabindex
   :Datatype:    Integer
   :Description: The index number for this element within a keyboard-based tabbing interaction sequence.
   :Standard:    NULL
   :Mandatory:   No

 - :Property:    onclick
   :Datatype:    String
   :Description: A piece of JavaScript code to be executed when the element is clicked or tapped.
   :Standard:    NULL
   :Mandatory:   No

 - :Property:    data
   :Datatype:    array
   :Description: Adds the "data-*" attribute to the tag. New since TYPO3 7 LTS. 
                 *Example:* `data="{foo:'bar'}"` produces `data-foo="bar"`
   :Standard:    NULL
   :Mandatory:   No
